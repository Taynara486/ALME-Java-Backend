# ALME — Java Backend

## Descrição

O ALME é um sistema backend desenvolvido em Java com o objetivo de identificar
produtos hypados de diferentes nichos e organizar essas informações para ajudar
revendedores a escolher produtos com maior potencial de venda.

O sistema busca estruturar dados de produtos, calcular tendência e organizar
listas com links de compra, facilitando a tomada de decisão.

## Problema

Revendedores perdem muito tempo escolhendo produtos e frequentemente compram
mercadorias fora de tendência, o que gera estoque parado e prejuízo.

A falta de análise de tendências torna o processo lento e pouco eficiente.

## Solução

O ALME propõe um sistema backend que identifica produtos hypados,
calcula o nível de tendência e organiza informações relevantes
para facilitar a escolha de produtos para revenda.

O sistema automatiza a análise e reduz o tempo de pesquisa.

## Motivação

O foco do projeto ALME não é apenas Programação Orientada a Objetos,
mas o domínio completo do backend em Java.

A proposta é aprender e aprofundar a linguagem Java de forma prática,
construindo um sistema real que resolve um problema existente no mercado
de revenda de produtos.

O projeto une aprendizado acadêmico, interesse pessoal em e-commerce,
prática de backend e construção de portfólio profissional.
